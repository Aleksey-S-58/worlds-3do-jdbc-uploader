This is a console application that allows to upload 3DO or Sprite to a database.

Package command contains commands to be executed to save 3DO or Sprite to a database.
Package config contains database configuration component.
Package exception contains uncheckable exceptions that allows to interrupt the process of data collecting and saving.
Package loader contains utilities allowing to read objects from files.
Package service.database.object.loader contains a service which is intended to save data to database.
